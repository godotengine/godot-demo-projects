# HDR Output

This project showcases high dynamic range output in Godot, as well as best practices.

See the
[HDR output documentation](https://docs.godotengine.org/en/latest/tutorials/rendering/hdr_output.html)
for more information.

Languages: GDScript

Renderer: Mobile
